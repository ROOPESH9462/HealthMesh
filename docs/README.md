# 🏥 AI Healthcare Management Platform

An AI-powered Healthcare Management Platform that helps hospitals, doctors, and patients manage healthcare digitally.

This platform combines Hospital Management System (HMS) features with Artificial Intelligence to improve patient care, automate medical workflows, and assist doctors in diagnosis.

---

# Main Features

- Patient Registration
- Doctor Dashboard
- Appointment Booking
- Electronic Medical Records
- Prescription Management
- Pharmacy Integration
- Billing System
- Video Consultation
- Hospital Analytics
- Disease Prediction
- Medical AI Assistant
- Medical Report Summarizer
- Prescription OCR
- CNN-based X-ray Disease Detection
